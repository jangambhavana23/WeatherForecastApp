# 🌦️ Weather Forecast Dashboard

A responsive Weather Forecast Application built using HTML, Tailwind CSS, Vanilla JavaScript, and OpenWeather API.

---

## Features

### Current Weather
- Search weather by city name
- Search weather using current location
- Display:
  - City Name
  - Current Date
  - Temperature
  - Humidity
  - Wind Speed
  - Weather Condition
  - Weather Icon

### Forecast
- 5-Day Weather Forecast
- Forecast Cards Display:
  - Date
  - Temperature
  - Humidity
  - Wind Speed
  - Weather Icon

### User Experience
- Responsive Design
- Mobile Friendly
- Tablet Friendly
- Desktop Friendly
- Loading Spinner
- Custom Error Messages
- Toast Notifications

### Additional Features
- Recent Search Dropdown
- Local Storage Support
- Temperature Toggle:
  - Celsius
  - Fahrenheit
- Dynamic Weather Backgrounds:
  - Clear
  - Clouds
  - Rain
  - Snow
  - Thunderstorm
- Extreme Temperature Alert

## API Used

OpenWeatherMap API

Base URLs:

Current Weather:

https://api.openweathermap.org/data/2.5/weather

Forecast:

https://api.openweathermap.org/data/2.5/forecast

## Installation

1. Clone repository

git clone YOUR_REPOSITORY_URL

2. Open project folder

3. Update API key inside:

js/app.js

Replace:

const API_KEY = "YOUR_API_KEY";

With:

const API_KEY = "YOUR_OPENWEATHER_API_KEY";

4. Run application

Open:

index.html

inside browser
